# **App Name**: FileFlipper

## Core Features:

- PDF to Word Conversion: Convert PDF documents into editable Word files.
- Word to PDF Conversion: Convert Word documents into PDF files.
- File Upload: Allow users to upload files from their computer or device.
- File Download: Provide a download link for the converted file.

## Style Guidelines:

- Primary color: Dark Indigo (#3F51B5) to evoke professionalism and trust.
- Background color: Very light desaturated Indigo (#F0F2F9) for a clean, uncluttered look.
- Accent color: Muted Blue-Violet (#5C6BC0) for interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, suitable for both headlines and body text
- Use simple, clear icons for file types (PDF, Word) and conversion actions.
- Clean and straightforward layout with clear sections for uploading, converting, and downloading files.
- Subtle loading animations during file conversion.
